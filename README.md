# fluxresume-cursor-mcp

Official Cursor plugin repository for [FluxResume](https://fluxresume.io).

## Plugins

| Plugin | Description |
|--------|-------------|
| [`fluxresume`](./plugins/fluxresume) | AI Agent — job tracking, job search, and career goal management via MCP |

## Install

Install via the Cursor plugin marketplace or point directly to this repository.

You will be prompted for your FluxResume MCP API key on install. Generate one in **FluxResume → Settings → AI Agent**.
